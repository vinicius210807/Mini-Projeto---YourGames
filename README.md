Projeto feito com: html, javascript e css

Descrição

O projeto se trata de uma plataforma onde o usuário pode registrar os jogos eletrônicos que ele jogou, podendo adicionar nota, imagem do jogo, plataforma onde foi jogado e status atual do jogo, podendo ser exibido estatísticas do total de jogos que o usuário jogou, finalizou ou platinou. O usuário pode fazer print da tela para compartilhar seus resultados com outros usuários já que a plataforma também possui um cadastro onde ele pode inserir uma foto de perfil e um nome de usuário que é exibido junto das estatísticas.

Conceitos Utilizados

Utilizamos as FUNCTIONS para executar os códigos como por exemplo o de inserir as informações do jogo na lista.
Utilizamos a LISTA para armazenar os jogos que o usuário inseriu;
Utilizamos o FOR para exibir os jogos na plataforma;
Utilizamos IF/ELSE para confirmar que os dados do jogo foram inserido e caso não fossem foi usado o ALERT para avisar o usuário.

Conceitos Utilizados que não foram vistos em aula
Utilizamos o HTML para escrever todo o conteúdo da plataforma e o CSS para estilizar a plataforma que foi utilizada em conjunto com o javascript para a parte de cadastro que quando o usuário termina o javascript usa o css para deixar invisível.
Utilizamos também uma fonte externa(através de um link rel) do google para estilizar todos os textos do site.

Algumas das tags utilizadas
<div> <img> <a> <button> <h1> <h3> <p> <input> <select> <option> <title> <link> 

IA Utilizada
